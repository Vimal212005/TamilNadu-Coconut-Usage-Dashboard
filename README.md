Tamil Nadu Coconut Usage Dashboard

Project Overview

This project is an interactive Power BI dashboard developed to analyze coconut consumption patterns and consumer behavior across Tamil Nadu using survey-based data collected through Google Forms.

The dashboard provides insights into:

* Coconut consumption trends
* Product preferences
* Regional usage distribution
* Occupation-based analysis
* Source and purchasing behavior
* Price inflation impact

The project was designed to demonstrate data visualization, dashboard storytelling, and analytical reporting using Power BI and DAX.

Key Insights

* Raw Coconut emerged as the most preferred coconut product.
* Retailers were identified as the most common coconut source.
* Students formed the majority of survey respondents.
* Family size showed a positive relationship with weekly coconut usage.
* Price inflation showed limited impact on the majority of respondents.
